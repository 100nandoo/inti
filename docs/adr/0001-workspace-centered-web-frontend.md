# Workspace-centered web frontend

Inti will keep one workspace page and simplify it with mode-driven `Input`, `Run`, and `Output` regions that act as UI groupings rather than replacing the existing product language of **Working Text**, **Transform Result**, **Result Surface**, and **Audio Result**. OCR staging and **Working Text** remain preserved across mode switches, speech generation runs from **Working Text** only, output selection is simplified around the relevant latest result, and promotion remains a single replace-only action because the older multi-surface layout and append-oriented controls no longer match the intended product shape.
